# Piyush Makhija — AI & ML Developer Portfolio

A modern, high-end, responsive personal portfolio website built with HTML5, CSS3, and JavaScript. This project showcases Piyush Makhija's profile, timeline education, technical skill metrics, featured engineering projects, IEEE research publications, certifications, and advanced interactive components in a polished, single-page layout.

🚀 Live Demo
See the live portfolio on GitHub Pages or your hosting provider after deployment.

✨ Advanced Interactive Features
- **AI Resume Assistant (Chatbot)**: Floating chat widget utilizing a client-side keyword and query parser with customized typing animations to answer questions about skills, projects, and contact info.
- **Project Architecture Deep-Dives**: Interactive cards that expand to show **Challenge → Solution → Impact** breakdowns, detailed pseudo-code implementations, and visual system data-flow diagrams.
- **3D Card Perspective Tilt**: Tactile, physical 3D perspective rotation on cards responding to cursor hover angles, integrated with moving gloss lighting reflections.
- **Skill-to-Project Matrix Links**: Mutual highlighting between technical skill pills and project cards during cursor hover.
- **Research Abstract Modal**: Overlay detailing the IEEE IC3 2025 cancer research paper abstract and key contributions.

🧩 Project Structure
- `index.html` — main portfolio page structure and schema markup
- `style.css` — high-fidelity cybernetic dark theme and responsive layout rules
- `script.js` — interactive behavior, cursor glow tracking, 3D tilts, chatbot, and modals
- `images/` — image assets, favicon, and generated abstract AI developer hero avatar

💻 Tech Stack
- HTML5
- CSS3
- JavaScript (ES6+)

🔧 How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/PiyushMakhija26/Portfolio-Website.git
   ```
2. Open `index.html` in a browser, or serve it locally:
   ```bash
   python -m http.server 8000
   ```

📦 Deployment
This repository is configured for easy deployment on static hosting platforms such as GitHub Pages, Netlify, or Vercel.
