IEEE IC3 2025 Abstract

Network Coding for Efficient Molecular Data Transmission in Cancer Research

Abstract

Molecular communication is an emerging paradigm where chemical molecules carry information. In medical applications, particularly distributed molecular biosensors in cancer diagnostics, transmitting biological telemetry reliably through turbulent body fluids faces severe bandwidth and noise constraints. This research introduces a network coding framework designed specifically for biological nano-nodes. By employing coding strategies at intermediate molecular nodes, the framework optimizes packet throughput and decreases transmission overhead.

Key Contributions

  • Network Coding Integration: Adapted traditional linear network coding algorithms to chemical diffusion channels, increasing transmission reliability.
  • Fault-Tolerance Optimization: Proved through stochastic channel modeling that network coding reduces transmission energy requirements by up to 35% compared to repeat-until-success protocols.
  • Simulation Testing: Validated communication trade-offs using bio-chemical simulation libraries, showing lower packet drop rates under turbulent flows.
